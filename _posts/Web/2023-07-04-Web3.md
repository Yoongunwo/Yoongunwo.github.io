## 1. Web3.0이란?
Web 3.0은 인공지능과 블록체인을 기반으로 맞춤형 정보를 제공하고 데이터 소유를 개인화하는 3세대 인터넷입니다. 기존에 사용하던 서비스에서 인공지능을 통해 개인 맞춤형 정보를 얻고, 블록체인을 통해 개인의 정보 소유 및 보안을 강화하는 지능형 웹 기술이다.

인공지능과 블록체인 등 최근 주목받고 있는 분야가 접목됨 웹 기술이기 때문에 Web 3.0 시대의 도래가 더욱 기대된다.
Web 1.0, Web 2.0을 통해 Web 3.0이 나온 배경과 블록체인을 중심으로 Web 3.0의 미래에 대해 자세히 알아보자
## 2. Web 1.0과 Web 2.0 세대별 특징
웹 기술을 시대적이로 구분하자면 과거에는 Web 1.0이 있었다. 현재는 Web 2.0의 시대이며, 앞으로 다가올 미래의 웹 기술이 Web 3.0인 것이다. 그렇다면 웹 기술은 그동안 어떻게 발전해 왔을까?

### 2.1 Web 1.0
Web 1.0은 '읽기' 중심의 초기 인터넷 기술이다. 과거의 웹 페이지는 소유자가 제공하는 정보만을 확인할 수 있다. 단순히 뉴스와 책 등을 인터넷으로 표현했다고 생각할 수 있는 시대이다.

### 2.2 Web 2.0
Web 2.0은 이전의 '읽기' 중심의 인터넷 기술에서 '쓰기'가 가능해졌고, 그에 따라 다양한 정보의 상호작용이 가능해졌다. 제공되는 플랫폼 안에서 우리는 다른 사람이 만든 정보를 확인할 수 있을 뿐만 아니라 정보를 직접 게시할 수도 있게 되었다.

웹상에서 다양한 사람과 양방향 커뮤니케이션이 가능한 Web 2.0은 시간이 지나며 점점 더 활성화됐다. 소셜 네트워킹, 게임 등 스마트폰으로 웹상에서 다양한 활동을 하는 우리가 살고 있는 현재가 바로 Web 2.0시대이다.

#### 문제점
- 웹의 핵심 가치인 개방성 저해
- 사용자 개인과 제작물 정보를 플랫폼이 소유
- 사용자 정보와 데이터로 플랫폼이 수익 창출 및 정보 통제
- 특정 플랫폼에서 개인정보 대량 유출 사건 발행

자유로운 커뮤니케이션이 가능한 Web 2.0은 정보 공유를 비약적으로 높여줬지만, 문제점도 있다. 바로 웹의 핵심 가치인 개방성이 낮아진 점이다.

Web 2.0에서는 우리가 만든 정보가 플랫폼에 의해 통제된다. 그리고 플랫폼은 사용자가 만든 정보와 데이터를 기반으로 수익을 창출한다. 구글이나 페이스북과 같은 거대 플랫폼 기업이 사용자의 정보나 제작물 정보를 통해 광고 상품을 판매하고 수익을 만드는 것처럼

이처럼 플랫폼이 소유한 사용자 개인 정보와 데이터를 이용해 광고 상품을 만들고, 상품 이용자는 플랫폼에 비용을 지불하고 광고를 진행하게 된다. 그렇지만 안타깝게도 Web 2.0은 정보의 원천인 개인에게는 그 수익이 돌아가지 않는 구조이다.

그리고 특정 플랫폼에 개인정보가 집중되면서 개인 정보가 유출되는 사건도 발생하기도 한다. 사용자가 많은 특정 플랫폼의 데이터베이스를 해킹하면 다수의 개인 정보를 얻을 수 있는 구조이이기 때문이다.

## 3. Web 2.0과 Web 3.0의 차이
Web 3.0은 정보의 읽기와 쓰기를 넘어 소유를 개인이 가능하게 만들어 개인화된 인터넷 환경을 구성하고자 한다. 따라서 Web 2.0과 Web 3.0의 가장 큰 차이는 정보를 누가 소유하느냐인 것이다. Web 2.0에서 플랫폼이 소유하고 있던 정보들을 Web 3.0에서는 개인이 소유하게 된다.

Web 3.0에서 '블록체인' 기술 통해 정보를 보호하고 개인이 정보를 소유하게 됨으로서 Web 2.0과 달리 보안성과 익명성이라는 장점을 얻을 수 있다. 현재 Web 2.0에서는 특정한 거대 플랫폼의 데이터 베이스를 공략해 다수의 개인정보를 탈취할 수 있다. 하지만 Web 3.0에서는 사용자 간에 직접 정보를 주고받는 P2P(Peer to Peer)가 가능해 정보를 탈취하려면 네트워크 전체를 탈취해야 하기 때문에 사실상 해킹이 불가능하다.

## 4. Web 3.0과 블록체인
Web 3.0 구성에 블록체인이 주목받는 이유는 정보의 암호화와 분산화를 통해 보안성을 높일 수 있는 장점때문이다. 앞서 언급한 사용자 간의 P2P는 블록체인으로 구성이 가능하며, 각 블록들이 서로 연결되어 있기 때문에 위변조가 어렵다. 그리고 블록체인 기술을 통해 플랫폼이 아니라 다시금 개인이 정보를 소유하게 되기 때문에 운영 주체가 플랫폼이 아닌 네트워크로 변경된다.
